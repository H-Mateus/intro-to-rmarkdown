Hear are examples of simple formating in Markdown. Note that nothing except the YAML header section (section 5) is specific to R Markdown.

1 Headings

We can create a heading with #, so for a level one heading we use:

  • # Heading level 1
  • ## Heading level 2
  • ### Heading level 3

and so on. If we have numbered sections turned on (as we do in on this page), these heading will be automatically numbered.

2 Simple text formating

As demonstrated in the prior page, we can wrap text with * to get bold and italic text:

  • bold text = **bold text**
  • italic text = *italic text*
  • bold and italic text = ***bold and italic text***

To start a new paragraph you need to leave a blank line.
